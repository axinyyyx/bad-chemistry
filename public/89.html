Actinium

Actinium is a radioactive element used in radiation therapy and as a neutron source in scientific research.

Atomic Structure

Atomic Information

Property Value
Atomic Number 89
Symbol Ac
Atomic Mass 227 u
Electron Configuration [Rn] 6d1 7s2
State at Room Temperature Solid

Uses of Actinium

  • Used as a neutron source in scientific research.
  • Used in radiation therapy for cancer treatment.
