Neptunium

Neptunium is a radioactive element used in the production of plutonium and for scientific research.

Atomic Structure

Atomic Information

Property Value
Atomic Number 93
Symbol Np
Atomic Mass 237 u
Electron Configuration [Rn] 5f4 6d1 7s2
State at Room Temperature Solid

Uses of Neptunium

  • Used as a precursor to the production of plutonium.
  • Employed in scientific research, particularly in nuclear physics.
