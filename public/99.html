Einsteinium

Einsteinium is a synthetic element named after Albert Einstein, primarily used for scientific research.

Atomic Structure

Atomic Information

Property Value
Atomic Number 99
Symbol Es
Atomic Mass 252 u
Electron Configuration [Rn] 5f11 7s2
State at Room Temperature Solid

Uses of Einsteinium

  • Used mainly in scientific research and synthesis of heavier elements.
  • Provides insights into the properties of transuranium elements.
